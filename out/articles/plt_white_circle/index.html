matplotlibで白抜き丸をプロットする

python, matplotlib

2021/06/27 01:34

plt.scatter で白抜き丸をプロットしたい...

python の plt.scatter で、散布図を描くことが出来ます。

from numpy.random import *
import matplotlib.pyplot as plt

x = randn(20)
y = randn(20)
plt.scatter(x, y, color="black")
plt.show()

上を実行すると、次の画像が生成されます。

これはこれでいいのですが、この丸を白抜きの丸にしてプロットしたいと考えた時があったので、備忘録として。
結論として、白抜き丸をプロットするには plt.scatter の引数にfacecolor = "None"を指定すれば良いです。

from numpy.random import *
import matplotlib.pyplot as plt

x = randn(20)
y = randn(20)
plt.scatter(x, y, color="black", facecolor="None")
plt.show()

実行すると次の画像が生成。 って感じです。
